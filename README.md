📝 **Sobre projeto**

Desenvolvi este site de notícias de tecnologia para aplicar meus conhecimentos em Flexbox e Media Queries. A interface conta com um sistema de navegação que permite ao usuário explorar diferentes seções e acessar conteúdos de forma responsiva.


### Tecnologias Utilizadas
1. HTML5
2. CSS3
3. JavaScript
4. Git e GitHub


### O que eu pratiquei?

Neste projeto, foquei em aplicar os seguintes conceitos:

* Flexbox;
* Media Queries;
* Gradiente de cores em CSS;
* Versionamento: Organização e histórico do código através do Git;
* Responsividade.
